# 03 内容范围

## 基本对象

LEO 的基本对象是人生中的伦理关系及其相关议题。

## 初步范围

LEO 初步包括但不限于以下伦理关系：

1. 个人与自身的伦理关系。
2. 个人与家庭成员的伦理关系。
3. 个人与伴侣、婚姻、亲密关系的伦理关系。
4. 个人与子女、父母及代际关系的伦理关系。
5. 个人与朋友、同伴和熟人的伦理关系。
6. 个人与职业、组织和劳动关系的伦理关系。
7. 个人与社区、社会和公共生活的伦理关系。
8. 个人与国家、法律和公共权力的伦理关系。
9. 个人与知识、教育、信息和表达的伦理关系。
10. 个人与技术、人工智能、数据和未来社会的伦理关系。
11. 个人与自然、生命共同体和未来世代的伦理关系。
12. 个人与苦难、死亡、信仰和终极意义的伦理关系。

## 范围调整

LEO 可以随着讨论和维护逐步调整内容范围。重大调整应说明调整理由。
