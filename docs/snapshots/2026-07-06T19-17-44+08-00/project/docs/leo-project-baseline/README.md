# LEO 项目基准

文档状态：第一版草案

项目名称：LEO

英文全称：Life Ethic Outline

中文名称：生活纲要

## 用途

本目录保存 LEO 的基础性项目文件。它说明 LEO 是什么、为什么存在、遵循哪些基本原则、当前覆盖哪些内容范围，以及版本和修订如何处理。

本文件不是生活纲要的数据主体。生活纲要的数据结构、条目体系、伦理关系分类和具体议题，应在其他文档中另行维护。

## AI 协作读取方式

为减少上下文消耗，AI 不应默认读取本目录全部文件。

推荐顺序：

1. 先读取 `index.json`，确认需要查看的章节。
2. 只读取与当前任务相关的章节文件。
3. 需要整体把握时，再读取本文件。

## 文件索引

- `00-meta.md`：项目基本信息和文件边界。
- `01-purpose.md`：项目使命。
- `02-principles.md`：基本原则。
- `03-scope.md`：内容范围。
- `04-versioning.md`：版本与发布。
- `05-revision.md`：项目基准的修订。
- `index.json`：面向 AI 和工具的章节索引。
