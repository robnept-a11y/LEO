# 02 基本原则

## 清晰性

LEO 的内容应尽可能使用明确、标准、普通人可以理解的简体中文表达。

## 开放性

LEO 应允许感兴趣的人阅读、讨论、引用或提出改进建议。

## 可追溯性

LEO 的正式内容形成、修改、删除和重组，应保留必要记录，使后来者能够理解其来源和理由。

## 多元性

LEO 应在合理范围内呈现和区分不同文化、宗教、哲学传统、社会经验和生活方式中形成的伦理理解。

## 非强制性

LEO 的内容不应被表述为对个人生活的命令，而应作为理解、判断和讨论伦理问题的辅助框架。

## 长期维护

LEO 不是一次性文本，而是可以随着社会变化、知识积累和讨论深化而持续修订的开放大纲。
