# 00 导言

## 定位

LEO 是一项长期维护的生活伦理知识项目，目标是形成并持续完善一份关于人生伦理关系及其相关议题的总提纲。

## 边界

本项目基准只规定 LEO 的基础定位、使命、原则、范围、版本发布和本文件自身的修订方式。

本项目基准暂不规定生活纲要的数据结构、维护角色、贡献流程、争议处理流程和具体伦理条目。

## 使命

LEO 的使命是：维护一份关于人生伦理关系和相关伦理议题的总提纲，使所有感兴趣的人能够通过网络服务方便地阅读、引用、讨论、修订和共享。

LEO 用于帮助人们识别伦理关系，理解伦理问题，比较不同立场，并在复杂人生情境中进行更清楚的思考。

LEO 不以替代个人判断、宗教信仰、法律制度、专业伦理规范或哲学体系为目的。

LEO 不宣称提供所有伦理问题的最终答案。对于存在合理分歧的问题，LEO 应尽可能呈现问题结构、主要立场、理由来源和争议点。

## 基本原则

### 清晰性

LEO 的内容应尽可能使用明确、标准、普通人可以理解的简体中文表达。

### 开放性

LEO 应允许感兴趣的人阅读、讨论、引用或提出改进建议。

### 可追溯性

LEO 的正式内容形成、修改、删除和重组，应保留必要记录，使后来者能够理解其来源和理由。

### 多元性

LEO 应在合理范围内呈现和区分不同文化、宗教、哲学传统、社会经验和生活方式中形成的伦理理解。

### 非强制性

LEO 的内容不应被表述为对个人生活的命令，而应作为理解、判断和讨论伦理问题的辅助框架。

### 长期维护

LEO 不是一次性文本，而是可以随着社会变化、知识积累和讨论深化而持续修订的开放大纲。
